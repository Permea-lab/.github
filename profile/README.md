# Permea Lab

**Open-source infrastructure for sequence-first therapeutic delivery and mRNA expression engineering.**

Permea Lab builds open benchmarks and AI tooling for designing, evaluating, and improving RNA and sequence-based therapeutic delivery systems.

## What we build

- **Permea Core** — reusable tooling for sequence-first delivery modeling and evaluation
- **Permea Signal ML** — machine learning benchmarks for mRNA expression and delivery signal prediction
- **Open benchmarks** — transparent evaluation tasks for bio-AI and biotech model development

## Focus areas

- mRNA expression engineering
- Sequence-first delivery optimization
- Bio-AI benchmarking
- Therapeutic delivery intelligence
- Open-source tooling for biotech AI
